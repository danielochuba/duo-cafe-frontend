# 📖 [DUO Cafe] <a name="about-project"></a>

**[DUO Cafe]** is a web application that allows users to add Cafe Item to cart, remove and purchase a cafe Item for a specific price, and for a lesser price if on discount. User can suscribe for news letter and recieve emails of new offers. User can also leave a feedback The page CRUD actions for (Category, Ingredients, Item, Offer, Subscriptions, Feedbacks) is performed by an Authorized ADMIN.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

Web Browser (Chrome recommended)<br>
Code editor (VS recommended)<br>
GitHub account<br>

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone "https://github.com/danielochuba/duo-cafe-frontend.git"
```
### Install

Go to your /duo-cafe-frontend folder:

```sh
  cd /duo-cafe-frontend
  npm install
  npm start
```
- SETUP [Back-end](https://github.com/danielochuba/duo-cafe-backend) 
Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone "https://github.com/danielochuba/duo-cafe-backend.git"
```

```sh
  cd /duo-cafe-backend
  bundle install
```	

### Usage

```sh
   rails s
```
